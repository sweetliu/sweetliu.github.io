ES2015

Posted on 2015-12-23   |   In FE

新的 ECMAScript 标准 ECMAScript 6（ES2015）在2015年6月发布，新语言借鉴了其他编程语言的优秀特性，宗旨是 Harmony，编程语言的和谐统一。
学习传送门：

  • babel
  • 给 JavaScript 初心者的 ES2015 实战

1. let+const

块级作用域变量，let是一个新的var，拥有块级作用域。const是定义一个常量，禁止修改

function f() {
  {
    let x;
    {
      // okay, block scoped name
      const x = "sneaky";
      // error, const
      x = "foo";
    }
    // okay, declared with `let`
    x = "bar";
    // error, already declared in block
    let x = "inner";
  }
}

2. 箭头函数=>和关键字this

箭头函数是函数的简写形式，这个在C#，Java 8和CoffeeScript中是很常见的。它支持表达式和语句体，和函数不同的是，箭头函数可以和函数以外的代码共享this

// Expression bodies
var odds = evens.map(v => v + 1);
var nums = evens.map((v, i) => v + i);

// Statement bodies
nums.forEach(v => {
  if (v % 5 === 0)
    fives.push(v);
});

// Lexical this
var bob = {
  _name: "Bob",
  _friends: [],
  printFriends() {
    this._friends.forEach(f =>
      console.log(this._name + " knows " + f));
  }
};

3. Class类

ES2015的类是一个基于原型的面向对象模式的简单语法糖。有一个简单的声明形式使得类模式更容易使用，并鼓励互操作性。类支持原型为基础的继承，超级调用，实例和静态方法和构造函数

class SkinnedMesh extends THREE.Mesh {
  constructor(geometry, materials) {
    super(geometry, materials);

    this.idMatrix = SkinnedMesh.defaultMatrix();
    this.bones = [];
    this.boneMatrices = [];
    //...
  }
  update(camera) {
    //...
    super.update();
  }
  static defaultMatrix() {
    return new THREE.Matrix4();
  }
}

4. 增强的对象字面量

对象字面量被扩展，支持在construction中设置原型，简写的语法形式foo:foo，定义super回调。对象字面量和class声明可以同时使用，让对象设计受益于一些相同的便利

var obj = {
    // __proto__
    __proto__: theProtoObj,
    // Does not set internal prototype
    '__proto__': somethingElse,
    // Shorthand for ‘handler: handler’
    handler,
    // Methods
    toString() {
     // Super calls
     return "d " + super.toString();
    },
    // Computed (dynamic) property names
    [ "prop_" + (() => 42)() ]: 42
};

5. 模板字符串

模板字符串为构造字符串提供了语法糖，这与Perl、Python或者其他语言的字符串插值特征相似。可选的，可以添加一个标记，以允许字符串结构来进行定制，避免从字符串内容注入攻击或构造更高级别的数据结构

// Basic literal string creation

var [a, ,b] = [1,2,3];

7. 函数参数

function f(x, y=12) {
  // y is 12 if not passed (or passed as undefined)
  return x + y;
}
f(3) == 15
function f(x, ...y) {
  // y is an Array
  return x * y.length;
}
f(3, "hello", true) == 6
function f(x, y, z) {
  return x + y + z;
}
// Pass each elem of array as argument
f(...[1,2,3]) == 6

9. 模块加载器 Modules

工程化过程中，各路大神总结了模块化构建的规则，CommonJS,AMD等，ES6总结了官方版的模块加载规则，一统江山。

// lib/math.js
export function sum(x, y) {
  return x + y;
}
export var pi = 3.141593;
// app.js
import * as math from "lib/math";
console.log("2π = " + math.sum(math.pi, math.pi));
// otherApp.js
import {sum, pi} from "lib/math";
console.log("2π = " + sum(pi, pi));
